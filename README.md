# TripRoad

여행상품 검색부터 예약, 결제 및 여행 준비까지 지원하는 통합 여행 플랫폼입니다.

## 주요 기능

- 지역·테마·상품명 복합 검색
- 여행상품 및 일정 조회
- 예약 및 카카오페이 결제
- 여행 캘린더와 체크리스트
- 찜 목록, 후기 및 1:1 문의
- 적립금 및 약관 관리
- 관리자 상품·일정·예약·회원 관리

## 기술 스택

- Frontend: React, TypeScript, Vite, Tailwind CSS
- Backend: Java, Spring Boot, JPA, MyBatis
- Database: Oracle Database
- Payment: KakaoPay API

## 실행 전 환경변수

DB_URL, DB_USERNAME, DB_PASSWORD, MAIL_USERNAME,
MAIL_PASSWORD, KAKAOPAY_SECRET_KEY 환경변수가 필요합니다.
